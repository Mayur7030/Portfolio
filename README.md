# Mayur Pawar — Senior Frontend Engineer Portfolio

Welcome to my personal portfolio repository! I'm a Senior Frontend Engineer with 4+ years of experience building scalable, high-performance enterprise web applications.

## 🚀 Live Demo

[https://mayur-pawar-portfolio.netlify.app/](https://mayur-pawar-portfolio.netlify.app/)

## ✨ Features
- **Modern UI/UX:** Premium aesthetic with a customized Teal/Amber color scheme, glassmorphism, smooth scroll reveals, and a custom tracking cursor.
- **Dark/Light Mode:** Fully functional theme toggler with `localStorage` state persistence.
- **Experience Timeline:** Detailed breakdown of my professional roles at Dassault Systèmes and Tata Consultancy Services.
- **Dynamic Projects Showcase:** Interactive portfolio gallery with category filtering (React, Full Stack, Clones) featuring projects like *Taskify*, *DevMatch*, and pixel-perfect clones.
- **Responsive & Accessible:** Built from the ground up to be fully responsive across mobile, tablet, and desktop displays.

## 🛠️ Built With
- **HTML5** semantic structure
- **CSS3** (Vanilla, CSS Variables for theming, Flexbox/Grid, Keyframe Animations)
- **JavaScript (ES6+)** (Vanilla JS, Intersection Observer API, DOM Manipulation)

## 📌 Sections Included
- **Hero:** Impactful introduction with animated statistics.
